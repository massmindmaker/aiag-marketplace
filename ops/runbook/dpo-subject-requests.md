# DPO — обработка запросов субъектов ПДн (C3)

## Legal basis

ст. 14, 20, 21 ФЗ-152 — субъект ПДн имеет право:
- получить подтверждение обработки + перечень его ПДн у оператора
- требовать уточнения / блокирования / удаления
- отозвать согласие

## Срок ответа

- **Доступ к информации** — 10 рабочих дней.
- **Уточнение / удаление / блокирование** — 30 календарных дней.

## Процедура

1. **Получение запроса** → email `dpo@ai-aggregator.ru` (alias → owner personal inbox).
2. **Регистрация** в Obsidian канбан `DPO requests` (board в `brain/Projects/AIAG/`).
3. **Identity verification:**
   - пользователь отправляет confirmation email с OTP, ИЛИ
   - нотариально заверенный запрос по почте (для крайних случаев).
4. **Action:**
   - Запрос на доступ → admin `/admin/users/:id` → `Export data` → signed S3 URL → email ответчику.
   - Запрос на удаление → `/admin/users/:id` → `Schedule deletion` (10 раб.дней grace period); после — hard delete.
   - Отзыв согласия → `/admin/users/:id#consents` → revoke; transborder revoke → auto-switch org в ru-only mode.
5. **Ответ** в канцелярском стиле (template ниже). Копия в `audit_log` (хранится 10 лет).

## SLA monitoring

Weekly review: все rows в `human_review_requests` со status=pending + deadline <7 дней → escalate primary. Row с status=pending AND deadline<now() → critical alert + email owner.

## Шаблоны ответов

### Шаблон 1 — доступ

```
Здравствуйте, [ФИО],

По вашему запросу от [DATE], ID #[...], направляем Вам:

1. Подтверждение обработки: Ваши персональные данные обрабатываются AI-Aggregator с [DATE].
2. Цели обработки: [список].
3. Перечень обрабатываемых данных: ФИО, email, телефон, IP-адрес, cookie, содержимое промптов.
4. Сроки хранения: до отзыва согласия + 3 года.
5. Выгрузка Ваших данных: [signed URL, действует 24h].

Оператор: ИП Боборов, ОГРНИП [...], реестр-номер РКН [...].
Ответственный за обработку ПДн: dpo@ai-aggregator.ru.

С уважением,
[ФИО DPO]
```

### Шаблон 2 — удаление

```
Здравствуйте, [ФИО],

Ваш запрос на удаление данных принят. Аккаунт запланирован к удалению на [DATE+10 рабочих дней].

До этого срока Вы можете отменить удаление в личном кабинете.

С уважением,
[ФИО DPO]
```

### Шаблон 3 — отзыв согласия на трансграничную передачу

```
Здравствуйте, [ФИО],

Ваше согласие на трансграничную передачу ПДн отозвано [DATE].

Ваш аккаунт переключён в режим «только РФ-хостинг» — доступны YandexGPT, GigaChat и другие модели на RF-инфраструктуре. Зарубежные модели (OpenAI, Anthropic, Fal) больше не вызываются.

Для восстановления полного каталога — `/account/settings#consents`.

С уважением,
[ФИО DPO]
```
